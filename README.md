# Java Refactoring Trial

## Dependencies

* Apache Maven 3.3.9
* Oracle JDK 1.8.0

## Compile

`mvn compile`

## Test

`mvn test`

## Run

From the candidate-app directory, run:

  `java -cp ./target/classes com.ibm.bluemix.CandidateApp`

or

  `mvn compile exec:java -Dexec.mainClass="com.ibm.bluemix.CandidateApp"`
